# Bean Life

## Usage

Running:

1. Install elm
2. `make`
3. open index.html in browser

Deploying:

1. Checkout gh-pages branch
2. Pull latest from master
3. Run `make`
4. Run `git add -f index.html`
5. Commit and push to gh-pages
